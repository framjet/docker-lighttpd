Lightweight HTTP Server Docker Image

Welcome! This is the default index page for the framjet/lighttpd Docker base image. This lightweight web server is preconfigured to serve static files from the /app directory and is ready for customization.

Base Image Features

  • Lightweight Lighttpd web server
  • Serves static files from /app directory (HTTP_ROOT)
  • Default MIME types configuration included
  • Health check endpoint at /status
  • Docker/Kubernetes liveness probe ready
  • Custom entrypoint scripts support: /docker-entrypoint.d/*.{sh,envsh}
  • Built-in wait4x support via WAIT_FOR_* and WAIT_CMD_FOR_* env vars
  • Optimized for static content serving
  • Low memory footprint and high performance
